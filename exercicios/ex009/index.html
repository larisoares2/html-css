Trabalhando com listas ordenadas e não ordenadas

Listas ordenadas

São listas que possurem uma ordenação, como numeros, portanto usamos a tag ol e dentro dela a li.

  1. Acordar
  2. tomar cafe
  3. trocar de roupa
  4. escovar os dentes
Para personalizar com letras ou numeros, podemos usar o type dentro do ol.

Listas não ordenadas

Não possuem ordenação, usamos a tag ul e dentro a li .

  • peito de peru
  • batata chips
Para personalizar o icone tambem utilizamos o type.
