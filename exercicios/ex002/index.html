Parágrafos e quebras de linha

Voçê pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags p e/p.

Observação: Para aparecerem os simbolos de maior e menor usados nas tags, voce pode utilizar a tag "e comercial" = "&+lt;" (less than) para abrir e "&+gt;" (greater than) para fechar. Ficando assim: <p> para abrir e </p> para fechar! Sem barra abre e com barra fecha.

Se precisar quebrar o texto em algum lugar especifico,
como este voce tambem pode usar a tag br, que significa break roll (quebra de linha)

Concluindo...

As tags <p> e </p> é usada para abrir e fechar paragrafos dentro de um texto, usando um espaçamento maior.

Já a tag <br> é utilizada para quebrar uma linha e não possui. fechamento

Para pular varias linhas não é recomendado usar varias tags <br>.

Adicionando simbolos e emojis no site

® = & reg
© = & copy
€ = & euro
enfim, é basicamente o simbolo de e comercial seguido do nome do simbolo, que pode ser achado no pdf ou no site Misc Symbols (https://www.w3schools.com/charsets/ref_utf_symbols.asp)

Para inserir emojis é usado o "ecomercial + # + x + codigo do emoji" para codigos hexadecimais e "ecomercial + # " para codigos decimais, como por exemplo:
hexadecimais 😄 🆓 🤓 🆓 🎄
para decimais 😔

nao copie e cole o emoji direto do site (http://emojipedia.org/) pq vai travar tudo
