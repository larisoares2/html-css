Tags para HTML5

Na linguagem HTML5, podemos utililizar os elementos da lista abaixo:

Passe o mouse sobre os termos para visualizar qual Tag utilizar

  • Título
  • Parágrafo
  • Linha horizontal
  • Lista ordenada
  • Abreviação com termo por extenso
  • Imagem
  • Favicon(acima do titulo)
  • Inserir link
  • Quebra de linha
  • Emojis
  • Citações espaçadas em outro parágrafo
